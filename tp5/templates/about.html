About This Application

Project Overview

This is a full-stack web application developed as part of TP5 - Docker exercise. It demonstrates the integration of multiple technologies and best practices in modern web development.

Technologies Used

Python 3.11

Backend programming language

Flask

Lightweight web framework

SQLAlchemy

SQL toolkit and ORM

Docker

Containerization platform

HTML5

Markup language

CSS3

Styling and layout

Features

  • User management system (CRUD operations)
  • Contact form with database storage
  • RESTful API endpoints
  • SQLite database integration
  • Responsive design
  • Flash messaging system
  • Docker containerization
  • Error handling (404 pages)

API Endpoints

GET /api/users

Returns all users in JSON format

GET /api/stats

Returns application statistics
